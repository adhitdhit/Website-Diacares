# DiaCARES - Diabetes Prediction System

Sistem deteksi dini risiko diabetes berbasis machine learning.

## Features
- Prediksi risiko diabetes real-time
- Menggunakan model XGBoost
- Database MongoDB Atlas
- Interface Gradio yang user-friendly

## Usage
Isi parameter pasien dan klik tombol "Prediksi" untuk melihat hasil.
